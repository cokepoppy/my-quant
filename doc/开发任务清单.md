## 技术注意事项

### 后端
- 使用 Prisma ORM 管理数据库
- 使用 BullMQ 处理异步任务
- 使用 Socket.IO 实现 WebSocket
- 使用 Winston 进行日志管理

### 前端
- 使用 Vue3 Composition API
- 使用 Pinia 进行状态管理
- 使用 Element Plus 作为UI组件库
- 使用 Chart.js/ECharts 进行数据可视化

### 数据库
- PostgreSQL 作为主数据库
- TimescaleDB 扩展处理时序数据
- Redis 用于缓存和任务队列
